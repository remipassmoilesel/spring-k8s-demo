Welcome !

Introduction

This is a small application built with Spring Boot and VueJS, intended to be deployed on a Kubernetes cluster. With this application, you can upload documents, sign them then check them with a PGP key.

How to use it

Upload and sign a document on page "New" (1Mo maximum).
See your document and the PGP signature associated on page "Documents".
Check a document against the uploaded one on page "Check". If the document is exactly the same, the check will be valid.

Which technologies are used here ?

For application backend:
  • Spring Boot as main framework
  • RxJava for reactive structure and performances
  • NATS messaging for synchronous communication between microservices
  • Helm Charts for Kubernetes deployments
  • Ansible for server configuration
For frontend:
  • VueJS as component oriented framework
  • Typescript as language
  • Webpack as application build tool
Development tools:
  • Docker Compose for development environment (should maybe replaced by Minikube)
  • JUnit 4, Mockito and Jacoco for testing and coverage reports (should be replaced with JUnit 5)
  • Gradle as application build tool
  • Python 3 for scripting
